# FrameBox 📷

FrameBox é uma solução self-hosted para gravação e visualização de câmeras IP (foco em iCSee/Yoosee), projetada para eliminar a dependência de serviços em nuvem pagos.

## Documentação de Projeto
- [Plano de Implementação](.antigravity/implementation_plan.md) (Contexto técnico e arquitetura)
- [Checklist de Tarefas (TASKS.md)](TASKS.md) (Acompanhamento do progresso)
- [Instruções para a IA](.instructions.md) (Regras de desenvolvimento)

## Tecnologias
- **Backend**: Node.js, TypeScript, Express, SQLite, FFmpeg.
- **Mobile**: React Native, Expo, TypeScript.

## Como começar
1. Siga a [Fase 1 do TASKS.md](TASKS.md) para configurar o ambiente.
2. Certifique-se de ter o `ffmpeg` instalado no seu sistema.
